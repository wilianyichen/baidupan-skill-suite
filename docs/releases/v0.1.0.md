# v0.1.0

发布日期：2026-04-27

这是 `baidupan-skill-suite` 的首个公开版本。

## 包含内容

- `baidupan-suite` 总 skill
- `bypy-enhanced` 浏览 / 搜索 / 上传下载
- `baidupan-monitor` 快照 / tree / compare
- `baidupan-reconcile` 本地 / 网盘、网盘 / 网盘差异对账
- `baidupan-fs` manifest 规划
- `baidupan-archive` 归档计划
- `baidupan-verify` manifest 验证
- `baidupan-apply` manifest 预览与执行
- `baidupan-index` 离线索引
- `baidupan-sync` 同步规划
- `baidupan-cleanup` 清理分析
- `baidupan-batch-runner` manifest 汇总与 runbook

## 关键能力

- 支持把整套工具作为单个 `baidupan-suite` skill 挂载给 Codex
- 支持 Linux 打包、安装和一键更新
- 支持录屏目录的结构比较与忽略扩展名对账
- 支持“归档优先”的安全执行链
- 支持目录树递归大小统计

## 发布产物

- Linux bundle：
  - `baidupan-skill-suite-v0.1.0-linux.tar.gz`

## 注意事项

- 发布包默认不包含 token
- 运行期目录、快照、虚拟环境均不进入 Git
- 如果从旧版升级，优先使用 `scripts/update_linux_bundle.sh`
