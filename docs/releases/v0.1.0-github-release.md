# baidupan-skill-suite v0.1.0

`baidupan-skill-suite` 的首个公开版本。

## Highlights

- Added `baidupan-suite` as a single Codex-facing entry skill
- Added Linux packaging, install, and one-step update scripts
- Added reconcile flow for local-vs-remote and remote-vs-remote comparisons
- Added archive-first execution chain:
  - plan
  - verify
  - preview
  - apply
- Added recursive directory size calculation for tree output

## Typical Workflow

1. Compare structures with `baidupan-reconcile`
2. Generate archive or filesystem manifests
3. Validate manifests with `baidupan-verify`
4. Preview manifests with `baidupan-apply`
5. Execute only with explicit confirmation

## Notes

- Release bundle does **not** include token files
- Runtime state such as `.venv`, `.bdpan_snapshots`, `.bypy_runtime` is not included in Git
- For upgrades on Linux, prefer `scripts/update_linux_bundle.sh`

## Upgrade

After uploading the new bundle and update script to Linux:

```bash
chmod +x ~/upload/update_linux_bundle.sh
bash ~/upload/update_linux_bundle.sh ~/upload/baidupan-skill-suite-v0.1.0-linux.tar.gz ~/tools
```
